Set up the Browser Agent

Download the Browser Agent extension and load it into your browser. Follow the steps below — it only takes a minute.

⬇ Download Extension (dist.zip)
⏳ Heads up: after you send a task, the Browser Agent can take 5–10 minutes to start working. Please be patient.

1 Download and extract the extension

  • Download the dist.zip file using the button above.
  • Right-click it → Extract All… → Extract (or use any unzip tool).
  • Open the extracted folder. You should see files like manifest.json, background.js, and sidepanel.js directly inside it.
⚠️ Important: the folder you select in Step 4 must be the one that contains manifest.json directly. If you opened the folder and see only another folder inside, go one level deeper until you see manifest.json.

2 Open the Extensions page

In your browser's address bar, type the matching address and press Enter:

BrowserAddress
Chromechrome://extensions
Edgeedge://extensions
Bravebrave://extensions

3 Turn on Developer mode

Flip the Developer mode toggle to ON (top-right of the Extensions page).

4 Load the extension

  • Click Load unpacked.
  • Select the extracted folder (the one containing manifest.json) and confirm.
  • Browser Agent now appears in your list of extensions. ✅

5 Pin it and start using it

  • Click the puzzle-piece 🧩 icon in the toolbar and pin Browser Agent for easy access.
  • Click the Browser Agent icon to open the side panel.
  • Wait for the panel to show Connected.
  • Type a task and send it.

Watch the agent work through the page step by step. It acts on the tab that's currently in focus, so keep the right tab active.
